Utimer dia fampiharana Android fameram-potoana tsotra iray hafa izay nataoko tamin'ny fanampian'ny React teratany.

## Maninona?

Tamin'izany fotoana izany dia nanontany tena aho hoe ahoana no hamoronana fampiharana Android ary nanapaka ny lohako tamin'ny tranokala mpamorona Android. Ny azoko lazaina dia ny fivoaran'ny Android miaraka amin'ny fiteny teratany (izany hoe Java sy Kotlin) dia tena mampatahotra ary tsy sariaka ho an'ny vao manomboka mihitsy.

Avy eo aho dia nanomboka nijery rafitra hafa toa ny React Native mba hamitana ny asa. Efa naheno momba ny React Native aho taloha, rafitra Javascript izay mamela anao hanangana fampiharana Android amin'ny alÃ lan'ny fampiasana ny paradigma React. Niharihary ny safidy satria efa nanana fahalalana momba ny React sy Javascript aho tamin'izany fotoana izany. Ary azoko lazaina fa nahavariana ahy ny maha-mora ny fanombohana amin'ny React Native sy ny maha-mora ny fanovana ny rindranasao amin'ny fampiasana ny React's JSX raha oharina amin'ny fampivoarana Android teratany sy XML ratsy tarehy.

## The process

Utimer dia fampiharana tsotra, ny lojikany rehetra dia ao anatin'ny singa tokana: [Main.js](https://github.com/radandevist/utimer/blob/master/src/pages/main/Main.js), ary tsy mampiasa API teratany ao anatiny. Ny hany nataoko dia ny nanao matematika isa fotsiny ary mampiasa ny fameram-potoana Javascript: `setInterval` sy `clearInterval` hanavao ny fanjakana sasany izay hampiasaina amin'ny fanivanana mifanohitra.

Fa kosa, ny fanamby lehibe indrindra dia ny fomba fanamboarana sy famelomana ny peratra fandrosoana mifanohitra, araka ny hitanao amin'ny fampisehoana gif teo aloha, miaraka amin'ny sandan'ny sata mifehy.

Mba hanaovana izany, dia nampiasa ny dingan'ny SVG boribory aho ary nahavita namatotra ny fananana style `strokeDashOffset` an'ity SVG ity miaraka amin'ny segondra sisa amin'ny kaontera. Azonao jerena ny zavatra resahiko ao amin'ny singa `<ProgressRing />` [eto](https://github.com/radandevist/utimer/blob/master/src/components/ProgressRing.js).
