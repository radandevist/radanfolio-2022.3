Samy manana Linux sy Windows (dual boot) installé amin'ny PC-ko aho, ka mizara telo ny disque dur:
- ny OS Linux,
- Windows 10, ary
- partition hizarana rakitra eo amin'ireo OS roa samihafa teo aloha ireo (satria tsy afaka mijery ny partition Linux avy amin'ny Windows isika).

Isaky ny miditra ao amin'ny Linux aho ary te hijery ny fizarako hafa dia tsy maintsy mi-monter azy ireo aloha aho. Saingy raha ny marina dia azontsika atao ny mi-monter azy ireo ho azy eo amin'ny boot na démarrage.

Sokafy ny fichier `/etc/fstab` miaraka amin'ny éditeur izay tianao:

```bash:terminal
sudo nano /etc/fstab
```

Hahita zavatra toy izao ianao (mety hiova kely izany arakaraka ny distro):

```bash:/etc/fstab
# /etc/fstab: static file system information.
#
# Use 'blkid' to print the universally unique identifier for a device; this may
# be used with UUID= as a more robust way to name devices that works even if
# disks are added and removed. See fstab(5).
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>                 
UUID=b57600bf-231d-4da2-bd44-e392eag85627 /              ext4    defaults,noatime 0 1
```

Araka ny hitanao dia mila ny UUID amin'ny fizarazarana tianao halefa ianao. Mba hahazoana izany, mandehana fotsiny amin'ny terminal:

```bash:terminal
sudo blkid
```

Ny lisitr'ireo fitaovana ao amin'ny masininao dia haseho miaraka amin'ny mombamomba azy, anisan'izany ny UUID-ny. Safidio ny UUID tianao ary avereno any amin'ny fichier `/etc/fstab`.

Any amin'ny faran'ny fichier ovay ny ligne:

```bash:/etc/fstab
UUID=YOUR_UUID /path/to/mount/point  auto,nofail,uid=1000,gid=1000,utf8,umask=022,defaults 0 0
```

Aza adino ny manolo uuid, mount point path ary ny partition filesystem.

### Tsara ho hay momba ny système de fichiers NTFS:

Izaho manokana dia te-hametraka fizarazarana azo jerena avy amin'ny Windows, izany hoe fizaràna <span class="text-brand1">ntfs</span>. Amin'ny Linux, ny lakile mifanaraka amin'ny fametrahana an'ity karazana système de fichiers ity ity dia ny `ntfs-3g`. Ny package mitovy anarana amin'izany no ilaintsika ao amin'ny système Linux antsika raha izany. Efa pré-installé amin'ny ankamaroan'ny distributions Linux izy io, fa jereo ihany raha sendra voafafa tsy nahy. Somary sarotra tamiko ny nikaroka an'io tamin'ny fotoana voalohany nampiasako ny Arch Linux, ka heveriko fa mety hanampy olona hafa ny milaza izany eto.

Tsy horesahiko ato amin'ity lahatsoratra ity ny options de montage rehetra, fa afaka mamaky bebe kokoa momba azy ireo ianao [eto](https://wiki.archlinux.org/title/fstab).
