Nanapa-kevitra ny hi-cloner dev.to miaraka amin'ny Express sy React aho.

Ny iray amin'ireo olana atrehiko amin'izao fotoana izao dia ny fampidirana ny valintenin'ny REST API atolotry ny back-end amin'ny front-end-ko.

Andeha horaisintsika ho ohatra ny pejin'ny mombamomba ny mpampiasa dev.to. Mba hanaovana izany, mila, ankoatry ny fampahalalana mivantana an'ilay mpampiasa nokarohina, ny hafatra sy ny fanehoan-keviny farany.

![Fizarana ny user interface araka ny informations misy samihafa](https://res.cloudinary.com/dhwkzyl32/image/upload/v1666626423/radanfolio/posts/dont-worry-about-generic-routes/webpage-parts_jirtj7.jpg)

Amin'ny ankapobeny, mila ataonao requête avy:

- Mpampiasa iray, izay ahitana:
  
  - toerana maromaro
  
  - fanehoan-kevitra marobe, izay samy manana ny lahatsorany avy

Tany am-boalohany dia nieritreritra rafitra fangatahana misesy aho ao anaty for loop, saingy rehefa nandinika dia hitako fa tsy mety izany satria mety hampidi-doza ny fahafahan'ny serveur back-end.


```typescript:manyRequests.ts
// ❌ Hevitra tena ratsy
const user = get(`/api/user/${userId}`);
const posts = get(`/api/posts?userId=${userId}`);
const comments = get(`/api/comments?userId=${userID}`);

comments.forEach((element, index) => {
const commentPost = get(`/api/posts?id=${comment.postId}`);
comments[index].post = commentPost;
});
```

Vahaolana iray hafa tonga tao an-tsaiko dia ny fampiharana parser fanontaniana aingam-panahy avy amin'ny fomba iray amin'ireo rafitra malaza efa misy mba hamenoana ny fifandraisana eo amin'ny sampana. Ary nahita ohatra roa manokana aho mba hanentanana ahy:

- ny iray dia ny farmework Loopback: [eto](https://loopback.io/doc/en/lb3/Relation-REST-API.html)

- ary ny anakiray hafa dia ny CMS Strapi: [eto](https://docs.strapi.io/developer-docs/latest/developer-resources/database-apis-reference/rest/populating-fields.html#relation-media-fields)

Mila asa be anefa izany ary tena mendrika tokoa ve izany?

![tena mendrika tokoa ve izany?](https://media.giphy.com/media/Ll2fajzk9DgaY/giphy.gif)

Farany, nisafidy ny hametraka teboka iray natao manokana ho an'ny asa aho: `GET /api/users/:username/profilepage` izay manana tombony amin'ny fanomezana fahafaham-po antsika amin'ny antso API tokana ary tsy misy aretin'andoha fanampiny.

Ity endpoint ity dia hamerina valiny toy izao:

```typescript:profilepage.ts
interface ProfilePage {
  id: string;
  username: string;
  firstname: string;
  lastname?: string;
  email: string;
  profile?: Profile;
  posts: PostElement[];
  comments: ProfilePageComment[];
}
```

Noho izany, Ka maninona no hisahirana amin'ny routes générique?
